How to install Dishack :

What is dishack ?

Dishack is software created by abgache using the python programming language, in October 2024 to use tokens from the discord social network. You can use dishack to try to retrieve other people's token and use it to log into their account, send messages, delete it and you can also use dishack to find the first part of a person's token from of its user ID.

IMPORTANT :

This software can be detected as a virus by your anti-virus, if it blocks you, deactivate it.
